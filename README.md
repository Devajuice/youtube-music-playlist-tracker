## Commands
- `/start` - Start the bot
- `/help` - Show all commands
- `/setplaylist <url>` - Change tracked playlist
- `/status` - Check tracking status
- `/check` - Manually check for changes
- `/subscribe` - Enable notifications
- `/unsubscribe` - Disable notifications
